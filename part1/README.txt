How to access Google's cloud NLP API for an existing project?

Steps (1) and (2) only apply to first timer.

(1) download and install Google Cloud SDK
(2) open Anaconda Prompt: pip install --upgrade google-cloud-language
(3) open Anaconda Prompt: set GOOGLE_APPLICATION_CREDENTIALS=[path for your credential json file]
(4) start jupyter notebook using Anaconda Prompt
